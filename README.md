# Arkanoid practice project

A classic Arkanoid-style breakout game built with Unity, part of challenge: https://20_games_challenge.gitlab.io/games/breakout/
Assets used: https://opengameart.org/content/breakout-brick-breaker-tile-set-free.
Paddle, ball, life, and background are Gemini generated.

## Overview

What I learned by building this game:
- Ball physics and paddle controls
- Play area boundary
- Brick destruction mechanics with scoring
- Lives system
- Scene Management - Return to game menu upon level completion
- C# events
- Scriptable Objects
- Singleton Pattern

## Installation via Unity engine
Clone the repository:
- git clone https://github.com/GoranK89/Arkanoid23112025.git
- Open the project in Unity
- Load the level1 or main menu scene and press Play
